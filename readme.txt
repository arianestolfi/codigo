# codigo
interface para digitalização da revista código

Interface para acesso público ao banco de dados da digitalização da revista código, protóptipo de um modelo de navegação experimental.

A finalidade do projeto é propor uma interface de navegação que fuja da lienaridade imposta pela materialidade do texto impresso, tendo como ponto de partida a digitalização da revista Código, de poesia concreta.

O modelo de navegação proposta permite diferentes percursos pelo material, com a possibilidade de livre recombinação das páginas ao mesmo tempo que permite a navegação tradicional de maneira linear pelas páginas duplas originais.




recursos implementados:
. navegação horizontal/vertical
. página simples quando vertical
. zoom
. página de busca
. endereço dinâmico
. integração odigo JSON
. conecção com collectiveaccess
. variação da largura das páginas
